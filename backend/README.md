# Expense Tracker App

A full-stack Expense Tracker built with React, Node.js, Express, MongoDB, JWT authentication, CSV upload/export, filtering, sorting, pagination, and charts.

## Tech Stack

### Frontend
- React + Vite
- Tailwind CSS v4
- Context API
- Axios
- Recharts
- React Router
- React Testing Library + Vitest

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT with HTTP-only cookies
- bcrypt password hashing
- Multer + CSV parser
- Swagger API documentation
- Jest + Supertest

## Features

- User signup/login
- JWT authentication
- Protected routes
- Add expenses
- Edit expenses inline
- Delete expenses
- Bulk delete expenses
- CSV bulk upload
- CSV export/download
- Search expenses
- Filter by category, date range, and payment method
- Sort by amount, date, and category
- Pagination
- Dashboard statistics
- Monthly line chart
- Category pie chart
- Payment method report
- Category auto-suggestions from previous entries
